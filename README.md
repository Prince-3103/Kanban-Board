#  Kanban Board

A modern Kanban Board built with HTML, CSS, and Vanilla JavaScript.

## Features

- Create Tasks
- Delete Tasks
- Drag & Drop Tasks
- Dynamic Task Counter
- Local Storage Persistence
- Responsive UI
- Modal-based Task Creation

---

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- Local Storage API
- Drag & Drop API
